Create Custom 3D Assets Using AI Meshy and Blender for Godot

Feb 15, 2025
5 min read

In this guide, we will go step by step to generate a 2D representation of a 3D object, convert it into a fully textured 3D model, and format it properly for use in Godot.

We will cover:

‚úÖ Using AI Image Generators to create a 2D image of your 3D object.

‚úÖ Removing the background and converting formats for best results.

‚úÖ Importing the 2D image into Meshy AI to generate a 3D model.

‚úÖ Fixing textures and exporting from Blender into Godot-compatible formats.

Step 1: Generate a 2D Image of Your 3D Object Using AI

To create a 3D model, we first need a high-quality AI-generated image that represents it.

üîπ Choosing an AI Image Generator

Some of the best free AI image generators include:

  • Leonardo.AI ‚Äì Best for detailed and stylized 3D-like renders.
  • Stable Diffusion ‚Äì Fully customizable for specific looks.
  • Dream by Wombo ‚Äì Easy and fast AI art generation.

üîπ Steps to Generate the Image

  1. Visit your chosen AI image generator .
  2. Enter a prompt describing your object. Example:

    ‚ÄúA futuristic sci-fi crate with glowing blue lights, 3D perspective, white background‚Äù

  3. Ensure the image is on a plain white background (this helps with background removal later).
  4. Download the generated image. Some AI tools may save it in WEBP format, which we will fix in the next step.

üìÇ Step 2: Convert WEBP to PNG (If Needed)

Many AI tools save images in WEBP format , but we need PNG for transparency support.

üîπ How to Convert WEBP to PNG

  1. Use an online tool like CloudConvert .
  2. Upload your WEBP image and convert it to PNG .
  3. Download the converted PNG file.

üßπ Step 3: Remove the Background Using Remove.bg

Now we need to remove the white background so Meshy AI can correctly detect edges and shapes.

üîπ Steps to Remove the Background

  1. Go to Remove.bg .
  2. Upload your PNG file .
  3. The tool will automatically remove the background .
  4. Download the new transparent PNG file .

üí° Why do this? AI models often blur the background into objects, so manually cleaning it up improves the 3D conversion.

üõ† Step 4: Import the Image into Meshy AI for 3D Conversion

Now that we have a clean 2D image , we will use Meshy AI to generate a 3D model with textures.

üîπ Steps to Convert the 2D Image to 3D in Meshy

  1. Visit Meshy AI .
  2. Click Upload Image and select your transparent PNG .
  3. Let Meshy automatically generate a 3D model .
  4. Click Download and select OBJ or FBX format .

Which format should you choose?

  • OBJ (.obj) ‚Äì Best for static models (no animations). This format is simple, widely compatible, and works well if you’re planning to manually assign textures in Blender or Godot. However, it doesn‚Äôt support animations or complex material settings.

  • FBX (.fbx) ‚Äì Better for animated models. FBX supports skeleton rigs, animations, and more complex material setups, making it ideal if your asset will have movement or animations.

üöÄ Meshy will also generate textures! These will be included as separate texture files.

üé® Step 5: Import the 3D Model into Blender for Cleanup

At this stage, the model might not be perfectly formatted for Godot , so we will:

‚úî Apply textures manually (if missing).

‚úî Convert the file format to .glb for best compatibility.

üîπ Importing the Model

  1. Open Blender and create a new project.
  2. Go to File > Import and select:
    • Wavefront (.obj) if Meshy exported an OBJ file.
    • FBX (.fbx) if Meshy exported an FBX file.
  3. Select the model and click Import .

üñå Step 6: Apply the Texture in Blender (If Missing)

Sometimes textures don‚Äôt automatically load , so we need to manually assign them.

üîπ Steps to Fix Textures

  1. Click on the model .
  2. Go to Shading Tab .
  3. In the Shader Editor , locate the Principled BSDF Shader .
  4. Press Shift + A ‚Üí Add “Image Texture” .
  5. Click “Open” and select the texture file that came with the model.
  6. Connect the Image Texture node to the Base Color input.

‚úÖ Your model should now display textures correctly!

üì¶ Step 7: Export to a Godot-Compatible Format

Now we need to convert the 3D model into a format that works perfectly in Godot .

üîπ Best Format for Godot: GLB (glTF 2.0)

  • Why GLB? ‚úÖ Loads faster than OBJ or FBX.

    ‚úÖ Supports textures, animations, and materials .

    ‚úÖ Native format for Godot , no extra steps needed.

üîπ How to Export as .glb in Blender

  1. Go to File > Export > glTF 2.0 (.glb).
  2. Choose:
    • ‚úÖ Format: glTF Binary (.glb) (Best for Godot).
    • ‚úÖ Transform: +Y Up (Godot uses Y as the up-axis).
    • ‚úÖ Include: Selected Objects.
    • ‚úÖ Geometry: Apply modifiers and keep materials .
    • ‚úÖ Embed Textures: Select “Export PBR” to keep materials in one file.
  3. Click Export .

üì• Step 8: Import the Model into Godot

Now we bring the optimized .glb model into Godot.

üîπ Steps to Import into Godot

  1. Open Godot and create a new project.
  2. Drag and drop the .glb file into your assets folder .
  3. Select the imported model and check the Inspector Panel :
    • Adjust materials if needed.
    • Make sure Textures are applied correctly.
  4. Click “New Inherited Scene” to place the model into your game.

‚úÖ Your AI-generated model is now fully in Godot! üéâ

üî• Final Recap: Complete Workflow

Step Tool Purpose
